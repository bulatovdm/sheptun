# Sheptun Roadmap

## Ближайшие улучшения

- [x] **Push-to-talk режим** — удерживаешь хоткей — идёт запись, отпускаешь — распознавание
- [ ] **Контекстные команды** — разные наборы команд для разных приложений (Terminal, VS Code, браузер)
- [ ] **Голосовая обратная связь** — озвучка подтверждений через TTS

## Средняя перспектива

- [ ] **Макросы и цепочки команд** — "сохрани и закоммить" → ⌘S, потом git commit
- [ ] **Улучшение распознавания** — пользовательский словарь, коррекция частых ошибок
- [ ] **Интеграция с Claude Code** — прямая отправка голосовых промптов

## Долгосрочно

- [ ] **Диктовка с пунктуацией** — "привет запятая как дела вопрос" → "Привет, как дела?"
- [ ] **Мультиязычность** — переключение языков голосом, смешанная речь

## Персонализация под пользователя (приоритет)

- [ ] **Fine-tuning Whisper под голос** — дообучение модели на записях пользователя для повышения точности
- [ ] **Пользовательский словарь** — автокоррекция частых ошибок распознавания (например "клот" → "клод")
- [ ] **Адаптивные пороги VAD** — автоматическая подстройка под громкость и тембр голоса
- [ ] **Сбор датасета** — запись и разметка голосовых команд для обучения
